# 📝 Task Manager Web App

## 📌 Project Description

This is a Task Manager web application built using HTML, CSS, and JavaScript.
It helps users manage daily tasks efficiently.

## 🚀 Features

* Add new tasks
* Display task list
* Mark tasks as completed
* Delete tasks
* Dynamic updates without page reload

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 File Structure

Task-Manager/
│── index.html
│── style.css
│── script.js

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open `index.html` in your browser
